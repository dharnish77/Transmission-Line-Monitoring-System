# ⚡ Transmission Line Monitoring System

An IoT-based smart monitoring system designed to continuously monitor transmission line health and detect faults in real time. The system helps reduce power outages by providing instant alerts and live monitoring through cloud connectivity.

---

## 📌 Project Overview

The Transmission Line Monitoring System is an intelligent embedded solution that monitors various parameters of high-voltage transmission lines. It detects abnormal conditions such as:

- 🌡️ Over Temperature
- ⚡ Voltage Fluctuation
- 🔥 Fire Detection
- 📏 Transmission Line Sag Detection
- 🚨 Tower Tilt Detection
- 💥 Fault Detection

Whenever an abnormal condition is detected, the system immediately sends notifications to the monitoring authority.

## 📡 Working Principle

1. ESP8266 continuously reads all sensor values.
2. Data is analyzed against predefined threshold values.
3. If any abnormal condition is detected:
   - Fault is identified.
   - Telegram alert is sent instantly.
   - Status is updated for remote monitoring.
4. Normal readings continue to be monitored in real time.
